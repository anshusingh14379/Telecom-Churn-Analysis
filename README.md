# 📊 Telecom Customer Churn Analysis

🚀 An End-to-End Data Analytics & Machine Learning Project focused on analyzing customer churn behavior, generating business insights, and predicting high-risk customers using SQL, Python, and Power BI.

---

# 📌 Project Overview

Customer churn is one of the biggest challenges faced by telecom companies. In this project, I analyzed telecom customer data to identify churn patterns, understand customer behavior, and build a Machine Learning model to predict customers who are likely to leave the company.

The project covers the complete analytics workflow:

✔️ Data Cleaning using SQL  
✔️ Exploratory Data Analysis (EDA)  
✔️ Data Transformation & Feature Engineering  
✔️ Machine Learning Model Building  
✔️ Churn Prediction  
✔️ Interactive Power BI Dashboard  
✔️ Business Insights & KPI Analysis  

---

# 🎯 Business Problem

Telecom companies lose revenue when customers leave their services. Identifying customers with high churn risk helps businesses take proactive retention actions.

This project helps businesses:

- Reduce customer churn
- Improve customer retention strategies
- Increase revenue stability
- Identify high-risk customer segments
- Make data-driven business decisions

# ⚙️ Project Workflow

## 1️⃣ Data Cleaning using SQL
- Handled missing values using `ISNULL()`
- Created clean production table
- Standardized categorical values
- Built SQL views for ML workflow

## 2️⃣ Exploratory Data Analysis
- Customer distribution analysis
- Revenue analysis
- Churn percentage breakdown
- Contract & internet service analysis

## 3️⃣ Machine Learning Pipeline
- Label Encoding for categorical columns
- Train-Test Split (80/20)
- Random Forest Classifier
- Model Evaluation using:
  - Confusion Matrix
  - Precision
  - Recall
  - F1-Score

## 4️⃣ Churn Prediction
- Predicted churn probability for newly joined customers
- Generated high-risk customer list

## 5️⃣ Dashboard Development
Created an interactive Power BI dashboard with:
- KPI Cards
- Customer Segmentation
- Revenue Insights
- Churn Analysis
- Dynamic Filters
- Business Decision Metrics

---

# 📈 Key Outcome of the Project

📌 The model predicted that **378 out of 411 newly joined customers** are at high risk of churn.

This insight can help businesses:
- Launch customer retention campaigns
- Improve customer engagement
- Provide personalized offers
- Reduce future revenue loss
